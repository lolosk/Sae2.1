Installation et Configuration de PostgreSQL sur Debian 10

1. Installation de PostgreSQL

sudo apt update
sudo apt install postgresql postgresql-contrib -y

2. Configuration de PostgreSQL

  • PostgreSQL est installé avec une configuration par défaut sécurisée.
  • Si nécessaire, ajustez les paramètres dans le fichier de configuration principal :
sudo nano /etc/postgresql/13/main/postgresql.conf
  • Redémarrez PostgreSQL pour appliquer les changements de configuration :
sudo systemctl restart postgresql

3. Accès à PostgreSQL

  • Connectez-vous à PostgreSQL en tant qu'utilisateur postgres :
sudo -i -u postgres
psql

4. Création d'une Base de Données et d'un Utilisateur

  • Créez une nouvelle base de données :
CREATE DATABASE nom_de_la_base;
  • Créez un nouvel utilisateur et attribuez-lui des privilèges sur la base de données :
CREATE USER nom_utilisateur WITH PASSWORD 'mot_de_passe';
GRANT ALL PRIVILEGES ON DATABASE nom_de_la_base TO nom_utilisateur;

5. Tests et Vérifications

  • Connectez-vous à PostgreSQL avec le nouvel utilisateur pour vérifier les accès :
psql -U nom_utilisateur -d nom_de_la_base -h localhost

6. Sécurité et Maintenance

  • Assurez-vous de sécuriser et de sauvegarder régulièrement votre base de données PostgreSQL.
