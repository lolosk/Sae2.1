Installation et Configuration d'un Serveur SFTP sur Debian 10

1. Installation d'OpenSSH (si ce n'est pas déjà fait)

sudo apt update
sudo apt install openssh-server -y

2. Configuration de SFTP

  • Assurez-vous que le service SSH est en cours d'exécution et accessible.

3. Utilisation de SFTP

  • Utilisez un client SFTP (comme FileZilla) pour vous connecter au serveur à l'aide de vos informations d'identification SSH.
