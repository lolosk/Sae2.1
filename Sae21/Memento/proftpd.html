Installation de ProFTPD sur Debian 10

1. Mise à Jour du Système

sudo apt update
sudo apt upgrade -y

2. Installation de ProFTPD

sudo apt install proftpd -y

3. Configuration de ProFTPD

a. Configuration de Base

  • Ouvrez le fichier de configuration principal de ProFTPD :
sudo nano /etc/proftpd/proftpd.conf
  • Ajoutez ou modifiez les options suivantes selon vos besoins :
# Activer le support IPv6 (optionnel)
UseIPv6 on

# Limiter ProFTPD à utiliser uniquement IPv4
UseIPv4 off

# Définir le masque d'UMASK par défaut pour les fichiers téléchargés
Umask 022

# Activer le mode passif et spécifier les ports
PassivePorts 49152 65534

# Définir le serveur par défaut et le masquer
ServerIdent on "FTP Server ready."
ServerName "Debian FTP Server"

# Utiliser l'authentification système pour les utilisateurs FTP
AuthPAM off

# Spécifier le dossier racine pour les utilisateurs FTP
DefaultRoot ~

# Limiter les utilisateurs FTP au dossier home
RequireValidShell off

b. Gestion des Utilisateurs FTP

  • Pour ajouter un utilisateur FTP :
sudo adduser nom_utilisateur
  • Pour changer le mot de passe d'un utilisateur FTP :
sudo passwd nom_utilisateur

4. Redémarrage de ProFTPD

sudo systemctl restart proftpd

5. Tests et Vérifications

  • Connectez-vous au serveur FTP avec un client FTP tel que FileZilla en utilisant les informations de votre utilisateur FTP pour vérifier la configuration.
