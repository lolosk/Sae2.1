Installation et Configuration d'un Serveur NFS sur Debian 10

1. Installation de NFS

sudo apt update
sudo apt install nfs-kernel-server -y

2. Configuration de NFS

  • Définissez les répertoires à partager en éditant le fichier d'export NFS :
sudo nano /etc/exports
  • Ajoutez une ligne pour chaque répertoire que vous souhaitez partager :
/chemin/vers/repertoire client_ip(rw,sync,no_subtree_check)

3. Redémarrage du Service NFS

sudo systemctl restart nfs-kernel-server

4. Configuration du Client NFS

  • Installez le client NFS sur les machines clientes et montez les partages NFS nécessaires.
