Installation et Configuration d'un Serveur SSH sur Debian 10

1. Installation de SSH

sudo apt update
sudo apt install openssh-server -y

2. Configuration de SSH

  • Ouvrez le fichier de configuration de SSH :
sudo nano /etc/ssh/sshd_config
  • Modifiez les paramètres selon vos besoins (par exemple, changer le port, activer/désactiver l'authentification par mot de passe, etc.) :
# Exemple de configuration minimale
Port 22
PermitRootLogin no
PasswordAuthentication yes
PubkeyAuthentication yes
AllowUsers votre_utilisateur

3. Redémarrage du Service SSH

sudo systemctl restart ssh

4. Connexion au Serveur SSH

  • Utilisez un client SSH comme PuTTY (Windows) ou Terminal (Linux/Mac) pour vous connecter au serveur.
