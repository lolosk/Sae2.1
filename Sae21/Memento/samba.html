Installation et Configuration d'un Serveur SAMBA sur Debian 10

1. Installation de SAMBA

sudo apt update
sudo apt install samba -y

2. Configuration de SAMBA

  • Configurez les partages en éditant le fichier de configuration de SAMBA :
sudo nano /etc/samba/smb.conf
  • Ajoutez des sections de partage pour chaque dossier que vous souhaitez partager :
[nom_partage]
   path = /chemin/vers/repertoire
   writable = yes
   guest ok = yes
   read only = no

3. Redémarrage du Service SAMBA

sudo systemctl restart smbd

4. Configuration des Permissions

  • Définissez les permissions appropriées sur les dossiers partagés pour les utilisateurs Samba.
